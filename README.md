# LAMPP-ARM64 STB
LAMPP Armbian 64 Bit | STB-B680H v2.1
## Download
```
git clone https://github.com/otn4mrehus/lampp-arm64v8.git
cd lampp-arm64v
```
## Run Container
```
sh service-lampp.sh
```

## Run Lampp
#### PHP 7.4
##### WEB
```
127.0.0.1:2000 
```
##### PHPMyAdmin
```
127.0.0.1:2001 
```
#### PHP 8.0
##### WEB
```
127.0.0.1:3000  
```
##### PHPMyAdmin
```
127.0.0.1:3001 
```
